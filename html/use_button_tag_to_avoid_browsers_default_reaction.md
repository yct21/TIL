# Use `button` tag to avoid browser's default action when clicked

Instead of using

- `<input type="submit">`
- `<button type="submit">`
- `<a href="javascript:void(0);">`

`<button type="button">` could be used when the browser is not supposed to do anything by itself when click occurred.
